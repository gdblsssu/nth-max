# Nth Max Project

Проект представляет RESTful сервис на Java (Spring Boot) для работы с Excel файлами, возвращающий N-е максимальное число из списка чисел.

## Требования
- JDK 17+
- Apache Maven
- Docker (опционально)
- IntelliJ IDEA
